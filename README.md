# Python Maze Game

Невелика гра-лабіринт, написана на Python з використанням Pygame.
Гравець керує персонажем та проходить через лабіринт до виходу.

## Стек технологій
- Python 3.14
- Pygame

## Встановлення та запуск
1. Клонуй репозиторій:
git clone https://github.com/taras-king/PMproject.git

2. Встанови залежності:
pip install -r requirements.txt

3. Запусти гру:
python main.py
